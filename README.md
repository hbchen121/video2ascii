# 视频转字符视频

把正常视频转化为字符视频。
具体原理见代码。

# 安装

保存本项目包本地，并使用 `python convertor.py -i xx.mp4` 进行转换。

# 环境配置

主要使用 moviepy 和 PIL 库

`pip install moviepy`
